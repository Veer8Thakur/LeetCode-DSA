Welcome to my central repository for tracking Data Structures and Algorithms (DSA) problems. This repository serves as a personal archive of optimized solutions, algorithmic patterns, and problem-solving strategies implemented during my daily coding practice.

## 🛠️ Automated Sync
The solutions in this repository are automatically tracked and pushed using **LeetHub v2** upon successful submission on LeetCode.

## 📚 Repository Structure

The problems are organized by their title and difficulty level. Inside each directory, you will find:
* **Problem Description (`README.md`):** The full problem statement, constraints, and examples scraped directly from LeetCode.
* **Solution File:** The clean, optimized source code file containing the accepted implementation.
